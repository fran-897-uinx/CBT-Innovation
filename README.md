
*TestprepAcademy (CBT-Innovation)*

TestprepAcademy is a comprehensive platform designed for seamless learning and examination. It features a sophisticated multi-role architecture and real-time collaboration tools.

*Tech Stack*
- Backend: Django (Python)
- Real-time: Node.js & Socket.io
- Database: PostgreSQL
- Payments: Paystack & Flutterwave

*Key Features*
- Multi-Role System: Tailored for Students, Tutors, and Directors.
- Automated Earnings: Custom commission engine (20% platform / 80% tutor).
- Real-time Chat & Video: Live learning sessions and group collaboration.
- Subscription Engine: Tiered access (Free, Pro, Premium) with expiry logic.
