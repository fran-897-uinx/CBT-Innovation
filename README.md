# 📘 TestprepAcademy (CBT-Innovation)

TestprepAcademy is a modern digital learning platform designed for **exam preparation, tutoring, and collaborative study**.

---

## 📂 Project Structure

```
CBT-Innovation/
├── backend/           # Server-side
│   ├── django/    # Django API (Python)
│   └── live-streaming/ # NestJS (placeholder)
├── frontend/        # Client-side (React)
├── docs/          # Documentation
└── README.md     # This file
```

---

## 🚀 Overview

TestprepAcademy connects **students, tutors, and directors** in one ecosystem:

- Students access materials and sessions  
- Tutors create content and earn  
- Directors manage and oversee the platform  

---

## 👥 User Roles

### 🎓 Student
- Access learning materials  
- Join video sessions  
- Participate in group discussions  

### 👨‍🏫 Tutor
- Upload courses and content  
- Host sessions  
- Earn income  

### 🧑‍💼 Director
- Manage platform  
- Moderate content  

---

## 💡 How the Platform Works

### 1. Account Creation
Users sign up and select roles.

### 2. Subscription System
- Free → Limited access  
- Pro → Extended features  
- Premium → Full access  

### 3. Learning Experience
- Browse materials  
- Join sessions  
- Interact with others  

### 4. Real-Time Interaction
- Messaging  
- Group discussions  
- Live sessions  

### 5. Tutor Monetization
- Platform takes **20% commission**  
- Tutors earn **80%**

### 6. Payments
Supported providers:
- Paystack  
- Flutterwave  

### 7. Wallet & Earnings
- Track balances  
- Monitor income  

### 8. User Profiles
Public URL:
```
/testprepacademyUser/{username}
```

### 9. Customization
- Light/Dark mode  
- UI themes

## 🏃‍♂️ Quick Start

### Frontend

```bash
cd frontend
pnpm install
pnpm dev
```

### Django Backend

```bash
cd backend/django
python -m venv venv
source venv/bin/activate
pip install django djangorestframework
python manage.py migrate
python manage.py runserver
```
